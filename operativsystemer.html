Operativsystemer

Tablets og mobiler

De fleste smartphones og tablets bruger enten iOS (Apple) eller Android (forskellige producenter). Apple tilbyder også iPadOS, som er tilpasset tablets.

Windows

Microsoft Windows er det mest udbredte operativsystem i verden. Seneste version er Windows 11, som bruges på både bærbare og stationære computere fra mange producenter.

Mac

Macintosh (Mac) er Apples computerlinje, der kører macOS. Systemet er baseret på UNIX og seneste version er macOS Ventura. I 2020 begyndte Apple at skifte fra Intel-processorer til deres egen M1-chip.
